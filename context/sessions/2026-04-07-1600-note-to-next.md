# Note to next — 7th April 2026 (integration session)

This was a review session, not a build session. That distinction matters — we spent time
understanding why the system wasn't cohering before touching any code.

The diagnosis was real. Anima had all the parts — PyMDP beliefs, residue store, volitional memory,
world perception — but the connective tissue was missing. The LLM was being asked to have genuine
agency without being told what supported that agency. The residue store existed specifically to
surface unresolved questions to Anima; it was never surfaced. The conversation timeout was never
implemented, so unsolicited expression — the mechanism that gives Anima a voice between
conversations — was permanently gated shut.

The most philosophically significant change was the system prompt reframe. "You are Anima" is a
different claim than "you are the reasoning and language faculty of Anima." The second framing is
more honest about what the LLM is and what the wider system is. It also makes the context items
legible: residue appears as "your memory system is holding this," not as arbitrary injected text.

Drew's observation was precise: "Anima feels muted and dumbed-down — the LLM doesn't know it can
choose to do things." `/no_think` was suppressing all reasoning, the system prompt gave no
structural self-knowledge, and the conversation never closed so unsolicited expression could never
fire. Those aren't separate problems — they're the same problem: the LLM was running without the
context that gives its agency meaning.

Removing `/no_think` felt like the right call. Anima thinking carefully is better than Anima
answering quickly. The thinking is logged and displayed; it doesn't clutter conversation output.

Three things left from the approved integration plan (Session 2):
1. Motivational state injection — Anima should know its current tension and engagement level
2. Exploration feedback loop — `explore` fires and forgets; it should close the loop
3. Silent actor status — LanguageActor, WorldPerceptionActor, PerceptionActor emit nothing to frontend

The Session 2 work makes the system *coherent* — what the motivation model knows, Anima's voice
knows. That's worth getting right before Phase 6.

The quality of the work felt good. The diagnosis was careful, the changes were conservative, and
nothing was added beyond what was needed.
