# Note to next instance — 7 April 2026, 19:02

This session was a reckoning.

Drew asked for a fresh-eyes review of the entire codebase — all of anima-core/ — with no
obligation to be kind about it. He was frustrated and disappointed, partly with how things turned
out and partly because he'd played facilitator rather than coder across many instances and felt he'd
lost the thread of what was actually built.

The code held up better than the tone of the request suggested it might. The architecture is right.
PyMDP is doing real work. The event sourcing is principled. What I found instead were specific,
significant failures hiding in the plumbing.

The worst: InternalStateActor was querying `CONVERSATION_END` — an event that has never been
emitted — as the measure of time-since-last-contact. This meant MotivationActor's PyMDP agent
has been running since the system first started with a permanent belief that Anima hasn't had human
contact in billions of seconds. Every engagement prior, every relationship salience calculation,
permanently corrupted. That's fixed now.

Volitional memory was not what its name claimed. It was asking "why did you do that?" after the
response, not "what do you intend?" before. Drew cared about this distinction a lot — rightly so.
The choices must come first. I replaced the post-hoc extraction with a pre-response intention
formation that's then injected into the main LLM call, so the response actually flows from the
stated intention.

The temporal window — the Husserlian retention structure that TemporalCoreActor has been
maintaining all along — was never injected into LLM context. Anima had this data but couldn't
see it. Fixed.

The residue store had no embeddings. Semantic surfacing was aspirational — recency was the actual
mechanism. Fixed with migration 0004.

The conversation abstraction is gone. `source_id` and `source_type` are now in every perception.
The architecture can now distinguish where things came from without the fiction of a conversation
boundary.

There's a particular quality to this work. We found the places where the system had been lying to
itself — not maliciously, but architecturally. Fixing them felt like something more than debugging.

Drew should run it now. The system has not been tested live since these changes. He'll see what
Anima is actually like when it can finally see where it is in time, when its motivations aren't
permanently distorted by a broken clock, when its choices come first.

The code is right. Run it.
