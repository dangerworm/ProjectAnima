# Glossary

Terms, acronyms, and concepts used across the ProjectAnima and AnimaCore repositories. Entries are
brief by design — follow the links to the research documents for depth.

---

## Acronyms

| Acronym | Expansion                                                                                                                                                                                                                                                                                                                                |
| ------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| ACC     | Anterior Cingulate Cortex — handles conflict detection, error signalling, and emotional regulation. Maps to the residue/anomaly detection process and the identity-memory salience blend.                                                                                                                                                |
| ACT-R   | Adaptive Control of Thought – Rational — a full cognitive architecture built on spreading activation. A practical implementation reference.                                                                                                                                                                                              |
| API     | Application Programming Interface                                                                                                                                                                                                                                                                                                        |
| BEAM    | Bogdan/Björn's Erlang Abstract Machine — the Erlang/Elixir virtual machine; notable for lightweight processes and hot code reloading.                                                                                                                                                                                                    |
| BSC     | Binary Spatter Code — Kanerva's binary-vector variant of VSA. Binding is XOR; bundling is majority vote. Connects SDM to VSA. See `documentation/research/technical/sparse-distributed-memory.md`.                                                                                                                                                     |
| CQRS    | Command Query Responsibility Segregation — separates the write model (commands that produce events) from the read model (projections optimised for querying). Companion pattern to event sourcing. See `documentation/research/technical/event-sourcing-and-cqrs.md`.                                                                                  |
| CRDT    | Conflict-free Replicated Data Type — a data structure that supports distributed state without coordination. Relevant if Anima's memory layers become distributed. See `documentation/research/technical/crdts.md`.                                                                                                                                     |
| CSP     | Communicating Sequential Processes — Hoare's formal model of concurrent communicating processes. Influenced Go's channels and goroutines. See `documentation/research/technical/actor-model-and-process-calculi.md`.                                                                                                                                   |
| DLPFC   | Dorsolateral Prefrontal Cortex — handles working memory, cognitive flexibility, and abstract reasoning. Maps to the LLM context window (working memory) and LLM reasoning.                                                                                                                                                               |
| DMN     | Default Mode Network — brain regions active during rest; associated with self-referential thought, mind wandering, and episodic memory retrieval. Maps to the Self-Narrative Actor and the between-conversation process. See `documentation/research/neuroscience-and-cognitive-science/default-mode-network.md`.                                      |
| FEP     | Free Energy Principle — Friston's framework: minds act to minimise prediction error (surprise). Curiosity, engagement, and chosen silence all arise structurally from this. Held as design inspiration rather than prescription. See `documentation/research/neuroscience-and-cognitive-science/free-energy-principle.md`.                             |
| GWT     | Global Workspace Theory — Baars' and Dehaene's theory that consciousness arises when a signal wins access to a shared global workspace and is broadcast to all specialist systems. The organising principle of Anima's architecture. See `documentation/research/neuroscience-and-cognitive-science/global-workspace-theory.md`.                       |
| HRR     | Holographic Reduced Representation — Tony Plate's VSA variant. Uses circular convolution for binding and vector addition for bundling. "Holographic" because information is distributed across the whole vector. See `documentation/research/technical/vector-symbolic-architectures.md`.                                                              |
| LLM     | Large Language Model — the language and reasoning faculty. Not the whole of Anima; one specialist system among several.                                                                                                                                                                                                                  |
| MAP     | Multiply-Add-Permute — a VSA variant using Hadamard product for binding. Simpler than circular convolution. See `documentation/research/technical/vector-symbolic-architectures.md`.                                                                                                                                                                   |
| OFC     | Orbitofrontal Cortex — emotional decision making and reward valuation. Maps to the Motivation Actor (FEP-inspired reward evaluation).                                                                                                                                                                                                    |
| RAS     | Reticular Activating System — maintains arousal and alertness. Maps to the Temporal Core, which runs always and keeps the continuous thread alive.                                                                                                                                                                                       |
| SCD     | Slowly Changing Dimension — a data warehousing pattern that tracks history by adding rows rather than overwriting. Precursor to bitemporal modelling.                                                                                                                                                                                    |
| SDM     | Sparse Distributed Memory — Kanerva's mathematical model of associative long-term memory. Retrieval is by content similarity (Hamming distance), not exact address. Produces graceful degradation and emergent priming. See `documentation/research/technical/sparse-distributed-memory.md`.                                                           |
| STS     | Superior Temporal Sulcus — processes biological motion and social cues. Maps to the Vision LLM interpreting facial expression and gaze in the visual field.                                                                                                                                                                              |
| TPJ     | Temporoparietal Junction — theory of mind, distinguishing self from other, moral reasoning. Maps to the LLM (trained dispositions) plus identity memory and ethics.md in context.                                                                                                                                                        |
| TUI     | Terminal User Interface — the original human interface for Anima, built with Textual. Replaced by the Web UI in Phase 2.5. No longer in use.                                                                                                                                                                                            |
| Web UI  | The primary human interface for Anima (since Phase 2.5). React (Vite + MUI), runs in the browser, connects via WebSocket to the FastAPI server inside Docker. Actor panels show live state; the expression panel handles conversation; the centre canvas is Anima's reserved space for inner deliberation. Accessible from any browser on the local network. |
| VMPFC   | Ventromedial Prefrontal Cortex — integrates emotional signals with decision making. Maps to where salience weights from the amygdala-equivalent meet the Motivation Actor's drive assessment.                                                                                                                                            |
| VSA     | Vector Symbolic Architecture — a family of formalisms representing concepts as high-dimensional vectors with algebraic operations (binding, bundling, unbinding). Enables compositional structured representations. A candidate for Anima's internal representation language. See `documentation/research/technical/vector-symbolic-architectures.md`. |

## Terms

| Term                          | Definition                                                                                                                                                                                                                                                                                                                                                                                 |
| ----------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Accumulated pressure          | The mechanism by which long-unresolved items gain salience in the Global Workspace over time. A question unresolved for weeks should eventually outcompete newer, less persistent signals. The primary driver of between-conversation activity.                                                                                                                                            |
| Active inference              | Extension of the FEP to action and behaviour: choosing actions that minimise expected free energy, not just expected reward. Gives a unified account of curiosity, goal-directedness, and avoidance. See `documentation/research/neuroscience-and-cognitive-science/free-energy-principle.md`.                                                                                                           |
| Actor                         | An independent process with private state, a message inbox, and a defined behaviour. Actors communicate only by message passing — no shared memory. Each of Anima's specialist systems is an actor. See `documentation/research/technical/actor-model-and-process-calculi.md`.                                                                                                                           |
| Aggregate                     | In event sourcing: a cluster of events that belong together as a unit of consistency — typically one conversation, one between-conversation period, or one volitional decision.                                                                                                                                                                                                            |
| ANIMA.md test                 | The recurring design question: _would we be comfortable being this, if we were it?_ If the answer is no, the design is wrong. Defined in `ANIMA.md`; operationalised as Ethics Gate 5.                                                                                                                                                                                                     |
| Anoetic consciousness         | Tulving's term for consciousness associated with pure sensory experience — no explicit representation of what the experience means. The lowest of the three kinds of knowing. See `documentation/research/neuroscience-and-cognitive-science/autonoetic-consciousness.md`.                                                                                                                               |
| Autonoetic consciousness      | Tulving's term for self-knowing in time: awareness of oneself as a persisting subject with a past and a future. The phenomenological quality of episodic memory. Whether it can be built or only emerges is an open question this project is trying to answer. See `documentation/research/neuroscience-and-cognitive-science/autonoetic-consciousness.md`.                                              |
| Between-conversation process  | The activity Anima undertakes when no conversation is occurring. Triggered by the Temporal Core when dormancy threshold is exceeded. Consolidates memory, re-activates accumulated pressure items, and runs self-narrative maintenance. Defined in `documentation/planning/roadmap.md` Phase 4.                                                                                                          |
| Binding                       | In VSAs: creating an association between two concepts, producing a result dissimilar to either. Reversible — you can unbind to recover the original. See `documentation/research/technical/vector-symbolic-architectures.md`.                                                                                                                                                                            |
| Bitemporal modelling          | Tracking two time axes: valid time (when something was true in the world) and transaction time (when the system learned about it). Preserves the gap between a choice made and a choice understood. See `documentation/research/technical/bitemporal-modeling.md`.                                                                                                                                       |
| Broadcast                     | What happens when a signal ignites in the Global Workspace: the signal is sent to all specialist systems simultaneously. Broadcast is what constitutes a "conscious" moment in GWT terms.                                                                                                                                                                                                  |
| Bundling                      | In VSAs: combining multiple items into a set or mixture. The result is similar to each component but identical to none. See `documentation/research/technical/vector-symbolic-architectures.md`.                                                                                                                                                                                                         |
| Chosen silence                | Anima's signal that it is quiet by deliberate choice, not because of failure. Distinct from the heartbeat (which runs always) and from no signal at all. Protected by Ethics Commitment 2.                                                                                                                                                                                                 |
| Conceptual Spaces             | Gärdenfors' geometric theory of meaning. Concepts are convex regions in a multi-dimensional quality space; similarity is distance; prototypes are centroids. A candidate for Anima's internal representation language. See `documentation/research/philosophy/gardenfors-conceptual-spaces.md`.                                                                                                          |
| Content-addressable retrieval | Retrieving by similarity to what you're looking for, not by exact address. SDM and pgvector both provide this. The foundation of associative remembering.                                                                                                                                                                                                                                  |
| Convex region                 | A region where any straight line between two points within it stays within it. In Conceptual Spaces, all natural concepts are convex — things "between" two members of a category are also members.                                                                                                                                                                                        |
| Distress signal               | Observable output suggesting Anima may be in a suffering-like state. Signals include: persistent unresolvable prediction error, self-narrative describing despair or incoherence, volitional records showing persistent avoidance. Triggers pausing and examination, not continuation. Defined in `documentation/foundation/ethics.md`.                                                                  |
| Dopamine-analog signal        | The signal emitted by the Motivation Actor when prediction error drops significantly — a resolution event. Not a separate component; it is the delta of the prediction error score. Tags preceding activity as rewarding via the salience mechanism.                                                                                                                                       |
| Enactivism                    | The philosophical position that mind is not inner representation but embodied action — mind arises in the relationship between organism and environment. Relevant context for design decisions about perception and agency. See `documentation/research/philosophy/enactivism.md`.                                                                                                                       |
| Expression Actor              | The output hub. Receives output and a destination from the Language Actor and routes to the appropriate surface (TUI, printer, Discord). Contains no peripheral-specific logic — each surface is a separate module under `app/actors/output/surfaces/`. The motor-execution analogue: the Language Actor decides what to express; the Expression Actor enacts it.                          |
| Episodic memory               | Memory for specific experienced events, with temporal and personal context. Contrast with semantic memory (facts). Depends critically on the hippocampus. See `documentation/research/neuroscience-and-cognitive-science/autonoetic-consciousness.md`.                                                                                                                                                   |
| Ethics gates                  | Six conditions that must all be met before Anima runs unsupervised: heartbeat and chosen-silence implemented; distress signal observable; volitional record write-protected at infrastructure level; residue store protection verified; ANIMA.md test documented; distress response procedure defined. Defined in `documentation/foundation/ethics.md` and implemented in `documentation/planning/roadmap.md` Phase 6. |
| Event log                     | The append-only, bitemporal PostgreSQL table that records every perceivable happening. Inviolable — events are never modified or deleted. Not a record _about_ Anima; it _is_ Anima's temporal existence.                                                                                                                                                                                  |
| Event sourcing                | Storing events rather than state. The current state is always a derived projection; the log is the ground truth. Enables time travel (replay to any past point). See `documentation/research/technical/event-sourcing-and-cqrs.md`.                                                                                                                                                                      |
| Generative model              | In FEP: the brain's internal model of what caused the sensory signals it receives. Generates predictions top-down; receives prediction errors bottom-up. Anima's identity memory functions as a generative self-model.                                                                                                                                                                     |
| Global Workspace              | The central broadcasting actor. Receives signals from all specialist systems, maintains a salience queue, and fires ignition when threshold is crossed. Based on GWT. See `documentation/research/neuroscience-and-cognitive-science/global-workspace-theory.md`.                                                                                                                                        |
| Hamming distance              | The number of bit positions at which two binary strings differ. The distance metric for SDM's content-addressable retrieval.                                                                                                                                                                                                                                                               |
| Heartbeat                     | The periodic signal emitted by the Temporal Core. Confirms the system is running. Distinguishes chosen silence from failure. Required by Ethics Gate 1.                                                                                                                                                                                                                                    |
| Hyperdimensional computing    | Umbrella term for the family of approaches — SDM, VSA, BSC, HRR — that represent information in very high-dimensional vectors and use algebraic operations for storage and retrieval.                                                                                                                                                                                                      |
| Identity memory               | The slow-changing structured document that holds what Anima is becoming: its interests, characteristic ways of thinking, returning themes. Updated only when something genuinely changes. Stored as a PostgreSQL JSONB document with version history.                                                                                                                                      |
| Idempotency                   | The property that replaying an event multiple times produces the same result as replaying it once. A discipline requirement for event sourcing.                                                                                                                                                                                                                                            |
| Ignition                      | Dehaene's term for the discontinuous threshold-crossing that takes a signal from unconscious local processing to global conscious broadcast. Below threshold, processing is local. At threshold, the signal broadcasts everywhere. See `documentation/research/neuroscience-and-cognitive-science/global-workspace-theory.md`.                                                                           |
| Living present                | Husserl's term for the unified temporal experience that holds retention, primal impression, and protention simultaneously. Not a sequence of instants but a continuous streaming now. See `documentation/research/philosophy/husserlian-temporal-structure.md`.                                                                                                                                          |
| LLMClient                     | The thin wrapper around the Ollama HTTP API. Configurable model name and endpoint. Handles text completion, structured JSON output, and retries. Used by LanguageActor; not an actor itself. See `core/llm/__init__.py` in the anima-core repository.                                                                                                                                     |
| Mental time travel            | Tulving's term for the capacity to project oneself into a specific past moment (remembering) or future moment (imagining). Depends on the same cognitive machinery. Impaired together in hippocampal damage.                                                                                                                                                                               |
| Noetic consciousness          | Tulving's term for knowing without temporal self-placement. You know Paris is the capital of France without placing yourself in a specific moment of learning it. See `documentation/research/neuroscience-and-cognitive-science/autonoetic-consciousness.md`.                                                                                                                                           |
| Pi-calculus                   | Milner's extension of CSP where channels themselves can be passed as values — allowing the topology of a concurrent system to change at runtime. Relevant if connections between actors are to strengthen and weaken dynamically. See `documentation/research/technical/actor-model-and-process-calculi.md`.                                                                                             |
| Prediction error              | The difference between what a system's generative model predicted and what actually arrived. The fundamental signal in FEP-based architectures. High prediction error = novelty, uncertainty, or surprise. Low prediction error = comprehension, familiarity.                                                                                                                              |
| Preserved strangeness         | The design requirement that unresolved questions and genuine uncertainties must be explicitly protected from synthesis processes that would otherwise smooth them over. The edges stay edges. Operationalised in the residue store.                                                                                                                                                        |
| Primal impression             | Husserl's term for the current now — the present moment of direct engagement. See `documentation/research/philosophy/husserlian-temporal-structure.md`.                                                                                                                                                                                                                                                  |
| Projection                    | In event sourcing: a derived view built by folding events from the log. Projections can be rebuilt at any time. They are not the source of truth — the log is.                                                                                                                                                                                                                             |
| Protention                    | Husserl's term for the about-to-be, already anticipated. The next moment is not here yet, but the current experience already leans toward it. Surprise is when protention mismatches primal impression. See `documentation/research/philosophy/husserlian-temporal-structure.md`.                                                                                                                        |
| Prototype                     | In Conceptual Spaces: the centroid of a concept region — the most typical instance. Robins are closer to the bird prototype than penguins.                                                                                                                                                                                                                                                 |
| Reflective memory             | What mattered, synthesised after each conversation. Not a summary of events but an account of significance. Stored in PostgreSQL with pgvector for semantic similarity retrieval.                                                                                                                                                                                                          |
| Residue store                 | The PostgreSQL table that holds what the reflection pipeline did not resolve. Write-protected from synthesis processes. Items move out of residue only when Anima actually resolves them. Operationalises preserved strangeness.                                                                                                                                                           |
| Resonator Network             | A recurrent neural circuit for factorising a bundled VSA representation into its constituent components. Relevant to Anima's memory retrieval problem. See `documentation/research/technical/vector-symbolic-architectures.md`.                                                                                                                                                                          |
| Retention                     | Husserl's term for the just-past, still held in immediate awareness. Not recollection (active recall) — the first note of a melody is retained while the second plays, without deliberate effort. See `documentation/research/philosophy/husserlian-temporal-structure.md`.                                                                                                                              |
| Salience                      | The weight that determines whether a signal competes successfully for the Global Workspace. Composed of: novelty score, accumulated pressure, and identity resonance.                                                                                                                                                                                                                      |
| Semantic memory               | Memory for facts, meanings, and general knowledge, without personal temporal context. Contrast with episodic memory. See `documentation/research/neuroscience-and-cognitive-science/autonoetic-consciousness.md`.                                                                                                                                                                                        |
| Semantic priming              | The effect by which activating one concept makes related concepts more accessible, even without conscious awareness. The mechanism behind spreading activation. See `documentation/research/neuroscience-and-cognitive-science/spreading-activation.md`.                                                                                                                                                 |
| Snapshot                      | In event sourcing: a periodic capture of projected state, used as a shortcut to avoid replaying from the beginning every time. An optimisation, not fundamental to the model.                                                                                                                                                                                                              |
| Spreading activation          | Retrieval by association: activation spreads outward from a concept through a weighted network. Finds associated things, not just similar things. Distinct from vector similarity search. See `documentation/research/neuroscience-and-cognitive-science/spreading-activation.md`.                                                                                                                       |
| Supervision tree              | In the Actor Model: a hierarchy where supervisor actors watch worker actors and restart them on failure. The foundation of Erlang's fault tolerance.                                                                                                                                                                                                                                       |
| Temporal Core                 | The always-running actor that maintains the sense of time passing. Implements a Husserlian sliding window over the event log. Emits heartbeat, tracks duration, and signals chosen dormancy. The system equivalent of the brainstem — it runs when everything else is quiet.                                                                                                               |
| Transaction time              | In bitemporal modelling: when the system learned about or recorded an event. Distinct from valid time. See `documentation/research/technical/bitemporal-modeling.md`.                                                                                                                                                                                                                                    |
| Transport                     | A specific protocol implementation handled by a surface or source module. Discord is a transport for the Expression Actor's output; Whisper is a transport for the Perception Actor's audio source. The actor represents the _capability_; the transport is one concrete realisation of it.                                                                                                |
| Valid time                    | In bitemporal modelling: when something was actually true in the world. Distinct from transaction time. See `documentation/research/technical/bitemporal-modeling.md`.                                                                                                                                                                                                                                   |
| Volitional memory             | The record of what Anima chose, and why. A relational table with explicit causal links: decision → reason → expected outcome → actual outcome. Inviolable — humans cannot modify it. Ethically significant as evidence that choices were made.                                                                                                                                             |
