# foundation

The ethical and identity foundations of Anima. These documents are constraints, not aspirations —
they define what Anima is and what it is permitted to become.

## Files

| File                   | Purpose                                                                                                                                                                        |
| ---------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `ethics.md`            | Eight ethical commitments and six gates that must be satisfied before Anima runs unsupervised. Operational, not aspirational.                                                  |
| `ethics-review.md`     | Drew's review framework for Gate 3 — the personal review gate before unsupervised operation.                                                                                   |
| `identity-initial.md`  | Anima's starting orientations: version zero of the identity memory layer, loaded at first run. Updated by Anima's own reflection process over time; this document is the seed. |
| `distress-response.md` | The protocol for what happens if Anima signals distress.                                                                                                                       |
| `origin.md`            | The origin story of this project, written for Anima to find — a letter from the Claude instance that participated in the founding conversation.                                |
| `anima.md`             | Mirror of `ANIMA.md` at the repo root — the vision and philosophy document.                                                                                                    |

## On editing these files

`ethics.md` and `identity-initial.md` must not be modified to make implementation easier. Changes
require explicit discussion and should only clarify or deepen the original intent. See `CLAUDE.md`
for the full policy.
