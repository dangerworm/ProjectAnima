# research

Background reading on neuroscience, philosophy, and technical topics relevant to Anima's
architecture. Organised into subdirectories by domain.

When a new field, technique, or topic comes up that is relevant to how Anima works or should be
built, a primer document belongs here and `reading-topics.md` should be updated.

## Files

| File                     | Purpose                                                                                 |
| ------------------------ | --------------------------------------------------------------------------------------- |
| `reading-topics.md`      | Index of all primer documents with brief descriptions.                                  |
| `cliffsnotes-version.md` | A condensed overview of key concepts across all domains — useful for quick orientation. |

## Subdirectories

### neuroscience-and-cognitive-science/

Theories from cognitive science that directly inform Anima's actor architecture: Global Workspace
Theory, autonoetic consciousness, the default mode network, the free energy principle, spreading
activation, and person/concept knowledge nodes.

### philosophy/

Philosophical frameworks relevant to Anima's nature and experience: enactivism, Gärdenfors
conceptual spaces, and Husserlian temporal structure.

### technical/

Implementation-relevant technical topics: active inference, the actor model, Apache Kafka,
bitemporal modelling, CRDTs, event sourcing/CQRS, sparse distributed memory, and vector symbolic
architectures.
